# Triplanar Unwrap

Automatically applies UV mapping by going through each mesh face by face, and then projecting UVs along the axis the face's normal is most simlar to.

![Triplanar Unwrap Panel](triplanar_unwrap.png)

The UVs are calculated as the world position multiplied by the Length factor and with the Offset factor added.

* Length - The stretching of UVs along each axis.
* Offset - An offset added to the UVs on each axis.
* Selected Faces Only - Only affect faces that have been selected in the face editor.
